AyameはオリジナルBootstrapテーマです

AyameはHonokaベースのBootstrapテーマの一つですが、以下の特徴を持っています。

Easy to Start

Honokaと同様AyameはBootstrapを元に製作されているため、非常に高い互換性を持っています。マークアップに関する規約はほとんど変わりません。

Replace Bootstrap

既にBootstrapを使って作成したウェブサイトがある場合は、CSSを置き換えるだけで簡単にAyameを利用できます。

Open Source

作成に使用したStylusファイルは全て公開されています。変数の定義ファイルを変更することで自分好みの設定に変更することも可能です。

Optimized Japanese

本家Bootstrapでは指定されていない日本語フォントに関する指定が行われているため、美しく日本語を表示できます。

Ayameは日本語を美しく表示します

本家Bootstrapでは指定されていないHonokaの日本語のフォント指定に加え、さらに調整した指定で日本語を美しく表示することに特化しました。

天下を我がものとすべく覇を競い合う権力者たちは、血で血を洗う争いに明け暮れていた。 誰もが己の欲にとり憑かれ、悪鬼の如き所業を繰り返す中、名君と呼ばれ、領民に慕われる領主は、郷田松之信のみであった。

だが、その松之信の下に、彼を援け、支え続ける二つの影があったことは誰も知らない。 闇に生き、闇に死すべき運命を負った、二つの影…。 東忍流の頭目、力丸と、同じくくノ一、彩女。 身も名も捨てて、己の腕を血で汚し、人知れず悪を誅する…。

これは、非道の世に終止符を打ち、松之信のため、領民のため、死と隣り合わせに生きた、二つの影の物語である…。
